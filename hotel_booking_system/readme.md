# 🏨 Hotel Booking System

A modern **Hotel Booking Management System** built with **HTML, JavaScript, Tailwind CSS**, and modular components.  
The system supports **authentication, booking management, content management, reporting, notifications, and more**.

## 🛠️ How to Open the Project

### 1. Extract the ZIP file  
- Download `project_folder.zip`.  
- Extract it → inside, you will find a folder named `hotel_booking_system`.  

### 2. Open in VS Code  
- Open **Visual Studio Code**.  
- Go to **File → Open Folder**.  
- Select the extracted `hotel_booking_system/` folder.  

### 3. Run the Project  
- Open any `index.html` file (for example: `Features/Auth/LoginPage/index.html`).  
- Right-click → **Open with Live Server** (recommended for smooth testing).  
- Or double-click the HTML file to open it directly in your browser.
